# TeleHealth+ — Demo Frontend

Plataforma web de telemedicina y bienestar digital

TeleHealth+ es una plataforma integral que reúne telemedicina, gestión de citas, historial clínico digital, tienda de productos de salud, gamificación y un asistente virtual llamado Healthy.
Este repositorio corresponde a la demo del frontend, desarrollada con tecnologías modernas.
No incluye backend, pero replica la experiencia visual completa del sistema.

Tecnologías utilizadas

- React
- Vite
- JavaScript ES6+
- Tailwind CSS
- React Router DOM
- Framer Motion
- Lucide React Icons
- CSS adicional

Backend original (no incluido)
- PHP
- Arquitectura SOA
- MariaDB / MySQL

Funcionalidades principales

Panel Administrativo
- Gestión centralizada de usuarios, citas, historial clínico, productos, pedidos, pagos, notificaciones, reseñas, blog y sistema de gamificación.

Panel Médico
- Herramientas para profesionales: seguimiento de pacientes, citas, emisión de recetas, mensajería y visualización de métricas de desempeño.

Panel del Paciente
- Acceso a citas médicas, videoconsultas, historial clínico, tienda virtual, pedidos, chat asistencial, notificaciones y puntos de gamificación.

Instalación y ejecución
1. cd frontend
2. npm install
3. npm run dev
